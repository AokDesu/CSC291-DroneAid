## One-time setup per developer

1. Clone the repo to your machine.
2. Set the env var `DRONE_AID_HANDLE` to your folder name above.
   - Windows (persistent): `setx DRONE_AID_HANDLE aok`
   - macOS / Linux: add `export DRONE_AID_HANDLE=aok` to `~/.zshrc` or `~/.bashrc`.
3. Add a `SessionEnd` hook to your **personal** Claude Code settings file
   (`~/.claude/settings.json` on macOS/Linux; `%USERPROFILE%\.claude\settings.json` on Windows):

   ```json
   {
     "hooks": {
       "SessionEnd": [
         {
           "matcher": "",
           "hooks": [
             {
               "type": "command",
               "command": "pwsh -ExecutionPolicy Bypass -File \"D:/projects/csc291/scripts/copy-claude-session.ps1\""
             }
           ]
         }
       ]
     }
   }
   ```

   On macOS / Linux replace the `command` with:
   ```
   bash /absolute/path/to/repo/scripts/copy-claude-session.sh
   ```

4. End a Claude Code session and verify a new `*.jsonl` appeared under `docs/agent-logs/<your-folder>/`.

## Redaction

Every session JSONL passes through `scripts/redact-secrets.py` before it lands here. The redactor strips:

- Google API keys (`AIza[0-9A-Za-z_-]{35}`)
- OpenAI/Anthropic-shaped keys (`sk-[A-Za-z0-9_-]{20,}`)
- OAuth tokens (`ya29\.[A-Za-z0-9_-]+`)
- Firebase service-account blobs (any JSON containing `"private_key"`)
- Bearer tokens (`Bearer [A-Za-z0-9._-]{20,}`)
- 13-digit Thai national IDs
- Email addresses

If you discover another secret-shaped string sneaking through, add a regex to `scripts/redact-secrets.py` in the same PR.

## Daily commit rule

Commit logs at least once per working day. Suggested cadence:

```bash
git add docs/agent-logs/<your-handle>/
git commit -m "logs: $(date +%Y-%m-%d) — <your-handle>"
git push
```

## CI gate

`.github/workflows/ci.yml` runs a `log-presence` job that fails any PR that:

- touches source files (anything outside `docs/` and `.github/`)
- by an author whose folder did not gain at least one new `.jsonl` in the PR diff.

That gate is the enforcement mechanism for the class rule.

## Auto-generated index

`docs/agent-logs/_index.md` is rebuilt by `.github/workflows/build-log-index.yml` whenever a new JSONL lands on `main`. Do not edit it by hand; your change will be overwritten on the next push.
